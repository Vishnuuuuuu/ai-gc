# AI Model Logos

Place the actual AI model logos in this directory.

## Required Logo Files

Download and place these logo files (PNG or SVG format, recommended size: 32x32px or larger):

### OpenAI Models
- `gpt.png` - OpenAI/GPT logo
  - Source: https://openai.com/brand/ (Official OpenAI brand assets)
  - Alternative: Search for "OpenAI logo PNG transparent"

### Anthropic Models
- `claude.png` - Anthropic/Claude logo
  - Source: https://www.anthropic.com/ (Anthropic brand page)
  - Alternative: Search for "Anthropic Claude logo PNG"

### Google Models
- `gemini.png` - Google Gemini logo
  - Source: https://gemini.google.com/ (Google Gemini brand)
  - Alternative: Search for "Google Gemini logo PNG"

### xAI Models
- `grok.png` - xAI/Grok logo
  - Source: https://x.ai/ (xAI official site)
  - Alternative: Search for "xAI Grok logo PNG"

## Quick Logo Sources

1. **Official Brand Pages** (Best quality):
   - OpenAI: https://openai.com/brand/
   - Anthropic: https://www.anthropic.com/
   - Google: https://about.google/brand-resource-center/
   - xAI: https://x.ai/

2. **Alternative Sources**:
   - Search "[model name] logo PNG transparent" on Google
   - Check company press kits
   - Use logoipsum.com as placeholders

## File Naming Convention

- All lowercase
- Use `.png` extension (or `.svg` for vector logos)
- Match the names exactly as shown above

## Fallback

If you don't have logos yet, the app will show text initials as fallback.
